RObrew
English | Português(Brasil)

O gerenciador de pacotes para Emulador de Ragnarok Online que faltava


Comando para instalar eAthena do repositório oficial. 
$ robrew install eathena
Comando para instalar rAthena do repositório oficial. 
$ robrew install rathena
Comando para instalar Hercules do repositório oficial. 
$ robrew install hercules
Listar e Instalar plugins do repositório robrew. 
$ robrew plugin list
$ robrew pluginstall plugin-name

outros emuladores serão adicionados o mais breve possíve. Por exemplo: brAthena, Cronus, etc.

Instalar RObrew
wget -q http://barrosfilipe.github.io/robrew/download/robrew && sudo mv robrew /bin/ && sudo chmod +x /bin/robrew

Cole isto em um Terminal.
