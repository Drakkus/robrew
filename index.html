The missing package manager for Ragnarok Online Emulator


Command to install eAthena from repository. 
$ robrew install eathena
Command to install rAthena from repository. 
$ robrew install rathena
Command to install Hercules from repository. 
$ robrew install hercules

other emulators will be added soon as possible. For example: brAthena, Cronus, etc.

Install RObrew
wget -q http://barrosfilipe.github.io/robrew/download/robrew && sudo mv robrew /bin/ && sudo chmod +x /bin/robrew

Paste that at a Terminal prompt.
